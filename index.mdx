Asciidoctor is a fast, open source, Ruby-based text processor for parsing AsciiDoc® content and converting it to HTML 5, DocBook 5, manual pages, PDF, EPUB 3, and other formats.

## What is AsciiDoc?

AsciiDoc is a lightweight, semantic markup language for writing technical content — articles, books, man pages, and more. It is readable in raw form and converts to polished, publication-ready output.

Asciidoctor is the processor: it reads the AsciiDoc source and converts it to your target format.

```asciidoc
= My Document
Author Name <author@example.com>
:toc:
:source-highlighter: rouge

== Introduction

This is a paragraph with *bold*, _italic_, and `monospace` text.

== Code example

[source,ruby]
----
require 'asciidoctor'

html = Asciidoctor.convert "Hello, *World*!", safe: :safe
puts html
----
```
