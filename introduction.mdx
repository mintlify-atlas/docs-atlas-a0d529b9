# Welcome to Asciidoctor

Asciidoctor is a fast, open source, Ruby-based text processor for parsing AsciiDoc® into a document model and converting it to output formats such as HTML 5, DocBook 5, manual pages, PDF, EPUB 3, and other formats.

AsciiDoc is the language. Asciidoctor is the processor.

## What is AsciiDoc?

AsciiDoc is a lightweight markup language for writing technical content. It is designed to be readable in raw form while also being convertible to publication-ready output formats. Asciidoctor is the successor to AsciiDoc.py and implements the AsciiDoc language.

```asciidoc
= My Document
Author Name <author@example.com>
:toc:

== Introduction

This is a paragraph with *bold*, _italic_, and `monospace` text.

== Code Example

[source,ruby]
----
require 'asciidoctor'
html = Asciidoctor.convert "Hello, *World*!", safe: :safe
puts html
----
```

## Asciidoctor Ecosystem

Asciidoctor is at the center of a rich ecosystem of tools and extensions:

| Project | Description |
|---------|-------------|
| [AsciidoctorJ](https://github.com/asciidoctor/asciidoctorj) | Run Asciidoctor on the JVM |
| [Asciidoctor.js](https://github.com/asciidoctor/asciidoctor.js) | Run Asciidoctor in JavaScript environments |
| [Asciidoctor PDF](https://github.com/asciidoctor/asciidoctor-pdf) | Convert AsciiDoc to PDF |
| [Asciidoctor EPUB3](https://github.com/asciidoctor/asciidoctor-epub3) | Convert AsciiDoc to EPUB 3 |
| [Asciidoctor Diagram](https://github.com/asciidoctor/asciidoctor-diagram) | Generate diagrams from text |
| [Asciidoctor Maven Plugin](https://github.com/asciidoctor/asciidoctor-maven-plugin) | Build integration for Maven |
| [Asciidoctor Gradle Plugin](https://github.com/asciidoctor/asciidoctor-gradle-plugin) | Build integration for Gradle |

## Supported Ruby Versions

Asciidoctor works on Linux, macOS, and Windows with the following Ruby implementations:

- **CRuby (MRI)** 2.7 – 3.4
- **JRuby** 9.4.0 – 10.0.2
- **TruffleRuby** (GraalVM)
